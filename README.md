# ft_containers
42 ft_containers project about some STL containers from C++98

What's done:
- list
- vector
- map
- stack
- queue
- allocator
- reverse_iterator
- iterator_traits and some type_traits adaptations
- pair and some relational operator templates

### Run tests
```clang++ main.cpp -std=c++98 -Wall -Wextra -Werror && ./a.out```
